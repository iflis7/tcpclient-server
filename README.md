A simple webserve and client side with C

### Running
 In the root folder run this command:
```
    make
```

## TCP Client:

 To run the client:
 ```
    ./tcpc <ip-address>
 ```

## TCP Server:

 To run the client:
 ```
    ./tcps
 ```
